# TextUtils.in ✨

TextUtils.in is a simple Django-based web application that allows users to analyze and transform text instantly. It provides useful text utilities like removing punctuation, capitalizing text, removing extra spaces, and removing new lines.

---

## 🚀 Features

- Remove Punctuations
- Capitalize Text
- Remove Extra Spaces
- Remove New Lines
- Clean and modern UI
- Instant text analysis
- Copy result to clipboard

---

## 🛠️ Tech Stack

- Python 🐍
- Django 🌐
- HTML5
- CSS3 (Modern UI with gradients & glassmorphism)
- JavaScript

---

## 📸 UI Preview

### Home Page
Users can enter text and select operations to perform.

### Result Page
Displays analyzed text with copy functionality.

## ⚙️ How to Run This Project

### 1. Clone the repository
```bash
git clone https://github.com/Sourabh108-Coder/TextFixer
cd mysite
```

### 2. Install dependencies
```bash
pip install django
```

### 3. Run Server
```bash
python manage.py runserver
```


## 🌐 Usage
1. Open browser and go to:
```bash
http://127.0.0.1:8000/
```
2. Enter text
3. Select operations
4. Click Analyze Text
4. Copy result easily
